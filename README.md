# Exercism Solutions

This repository contains my solutions to various programming exercises from [Exercism](https://exercism.org/). Exercism is a platform that helps you improve your coding skills through practicing with small, focused exercises in multiple programming languages.

## Structure

The solutions are organized by programming language and then by exercise name:

- `solutions/javascript/` - JavaScript solutions
- `solutions/typescript/` - TypeScript solutions

Each exercise folder contains one or more versions of the solution, numbered sequentially (e.g., `1/`, `2/`, `3/`).

## Languages

- **JavaScript**: Solutions for JavaScript exercises
- **TypeScript**: Solutions for TypeScript exercises

## Getting Started

To run any of these solutions:

1. Make sure you have Node.js installed
2. Navigate to the specific exercise folder
3. Run the JavaScript/TypeScript file using Node.js

For example:

```bash
cd solutions/javascript/hello-world/1
node hello-world.js
```

## Contributing

This is a personal learning repository. Feel free to explore the solutions, but keep in mind that there are often multiple ways to solve each exercise!
